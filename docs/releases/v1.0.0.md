# SuperNoNo v1.0.0

SuperNoNo v1.0.0 is the first public release candidate for the desktop agent
companion.

## Highlights

- Compact Electron desktop pet UI with transparent always-on-top window.
- Minimal pet mode with bottom status bar and speech bubble.
- Tray menu for show/hide, task panel, settings, demo, and quit.
- Hover interaction for lightweight ambient pet motion.
- Local signal bridge on `127.0.0.1:4174`.
- Agent-neutral signal protocol for Codex, Claude Code, Cursor, generic CLI, and
  future adapters.
- Verified Codex Desktop plugin hooks:
  - `command_running` from `PreToolUse`
  - `step_done` from `PostToolUse`
  - `turn_ended` fallback from notify wrapper
- Codex adapter/plugin files included in the repository.

## Windows Portable Build

Download `SuperNoNo-win32-x64-v1.0.0.zip`, unzip it, and run:

```text
SuperNoNo.exe
```

This is a portable package. It does not install shortcuts or auto-updates.

## Known Notes

- Codex plugin hook setup still requires local configuration and hook trust.
- Windows `command_windows` in the Codex plugin currently uses a machine-specific
  Node path and should be productized in a future setup script.
- `legacy_notify` Windows error 206 is tracked as a separate fallback issue.
